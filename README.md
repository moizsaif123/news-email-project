# News Email Project

This Python project fetches top news headlines from NewsAPI.org and emails them using Gmail SMTP.

## Main File

`news_email.py`

## Requirements

Install packages:

```bash
pip install -r requirements.txt
```

## Secrets Needed

This project needs:

- `NEWSAPI_KEY`
- `GMAIL_ADDRESS`
- `GMAIL_APP_PASSWORD`

Do not upload `.env` to GitHub.
